# ReSyOn

ReSyOn es una aplicación creada para agilizar las requicisiones de una empresa.

### Requisitos

* Ruby 2.2
* Rails 4.1.6
* RubyGems 2.4.6

### Instalación

```sh
$ git clone clone https://github.com/Metuz/compras
$ bunble install
$ rake db:setup
$ rails s
```
